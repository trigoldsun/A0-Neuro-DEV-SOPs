## 3. 项目级凭证

各项目入驻后自动生成的独立凭证保存在：

```
/opt/neuro-devsituat-platform/.env.secrets
```

查看方式：

```bash
cat /opt/neuro-devsituat-platform/.env.secrets
```

每个项目会生成：

| 凭证 | 格式 |
|:---|:---|
| 数据库用户 | `<项目编号>_user` |
| 数据库名 | `<项目编号>_db` |
| 数据库密码 | 24位随机字符串 |
| Redis ACL用户 | `<项目编号>` |
| Redis密码 | 24位随机字符串 |
| MinIO桶名 | `<项目编号>` |

### 项目连接串模板

```
# PostgreSQL
DATABASE_URL=postgres://<项目编号>_user:<项目DB密码>@postgresql18:5432/<项目编号>_db?sslmode=disable

# Redis
REDIS_URL=redis://:<项目Redis密码>@redis7:6379/0

# MinIO
MINIO_ENDPOINT=minio:9000
MINIO_ACCESS_KEY=minioadmin
MINIO_SECRET_KEY=TUewRmZakDzlFB5POYcGi0bA
MINIO_BUCKET=<项目编号>
```
